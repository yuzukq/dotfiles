# VRMモーション設計リファレンス

spec を設計する前に必ずこのファイル全体を読むこと。

# 座標規約 (VRM 1.0 / T-pose レスト)
- モデルは +Z を向いて立つ。+X はモデルの左手側、+Y が上。
- 回転はレストポーズ(Tポーズ)からのオイラー角 [X, Y, Z] (度、XYZ順)。
- 腕はTポーズで真横に伸びている:
  - 左腕を下ろす: leftUpperArm Z=-70 / 右腕を下ろす: rightUpperArm Z=+70
  - 右腕を高く上げる: rightUpperArm Z=-60 前後 / 左腕: Z=+60 前後
  - 肘を曲げる: leftLowerArm / rightLowerArm を回転 (右肘を曲げて手を上げる: rightLowerArm Z=-90 付近)
  - 腕を前に出す: leftUpperArm Y=-60 / rightUpperArm Y=+60
- 前屈・うなずき: spine/chest/neck/head の X を正方向 (+20 で前へ 20 度)
- 頭を左右に向ける: head Y (正 = モデルの左を向く)
- しゃがむ: hips の p.y を負に + leftUpperLeg/rightUpperLeg X=-45, leftLowerLeg/rightLowerLeg X=+80 など
- ジャンプ: hips の p.y を一時的に +0.2〜0.3
- 体全体の向き変更は hips の Y 回転

# 使用可能なボーン
hips, spine, chest, upperChest, neck, head,
leftShoulder, leftUpperArm, leftLowerArm, leftHand,
rightShoulder, rightUpperArm, rightLowerArm, rightHand,
leftUpperLeg, leftLowerLeg, leftFoot, rightUpperLeg, rightLowerLeg, rightFoot
(※ shoulder と hand は指定しても自動処理に置き換えられる。下記ルール参照)

# 表情 (expressions)
- 使える表情: happy, angry, sad, relaxed, surprised, neutral, aa, ih, ou, ee, oh,
  blink, blinkLeft, blinkRight, lookUp, lookDown, lookLeft, lookRight
- w はウェイト 0〜1。感情表情は 0.4〜1.0、変化には 0.2〜0.4 秒かける。
- モーションの感情に合った表情を入れると生きる (喜ぶ→happy、落ち込む→sad、驚く→surprised)。
- まばたき (blink) を 2〜4 秒おきに: 0→1→0 を約 0.15 秒で。

# ルール
- 常に腕を下ろした自然な姿勢から始める (leftUpperArm Z=-70, rightUpperArm Z=+70 を t=0 に置く)。
- 使うボーンには必ず t=0 と t=duration のキーを置き、非ループなら最初と最後をニュートラルに戻す。
- キーは線形+球面補間される。緩急をつけるためキーを十分に打つ (目安: 1秒あたり2〜4キー)。
- duration: 単発動作 (うなずく・手を振る) は 2〜4 秒 / 連続演技・ダンスは 8〜15 秒でフェーズ分け。最大20秒。
- leftHand / rightHand (手首) は動かさない (自然な手ポーズが自動適用)。
- leftShoulder / rightShoulder も動かさない (腕上げに自動追従、指定しても削除される)。腕は upperArm で。
- 肘 (lowerArm) は Z (または Y) を主軸に。X は使わない (前腕が破綻する。バリデーションで ±10 に制限)。
- 首・頭の合計は ±60 度以内。spine/chest はそれぞれ ±30 度以内。
- 動きの主役となる関節を決め、それ以外は控えめに。

# 自然なポーズの原則
- 腕上げは斜め上まで: upperArm Z は右 -75〜-40 / 左 +40〜+75。真上 (±90) 超えで頭に被せない。
- 立ち姿勢では肘に 10〜30 度の曲げを残す (伸びきった腕はロボットに見える)。
- 寝転ぶ・倒れる姿勢では逆に腕を床に沿わせ、肘は 0〜15 度。
- 往復運動は端で減速 (折り返し直前に中間キー)。等速往復は機械的。
- 動き出しに小さな予備動作 (0.1〜0.2秒の逆方向の溜め)、終わりに余韻。
- 大きな動作には head や chest の小さな傾き (5〜10度) を連動させる。

# 頻出動作の解剖学メモ
- 手を振る: 上腕は横斜め上 45〜60 度 (rightUpperArm Z=-45〜-60)。肘を 60〜90 度曲げて
  手を「顔の横〜頭の高さ」に置き、前腕 (lowerArm Z) を往復。例: rightUpperArm [0,0,-50] 維持、
  rightLowerArm [0,0,-40]⇔[0,0,-55] 往復。腕を真上に伸ばして振るのは不自然。
- 上腕を 60 度以上上げた状態で肘を 60 度以上曲げない (前腕が頭に被さる。自動補正あり)。
- 拍手: 両腕を体の前 (upperArm Y ±50〜60 + 少し下げ)、肘から先を小さく開閉。
- 膝 (lowerLeg) は蝶番: X の 0〜130 のみ。逆関節は禁止 (自動クランプ)。
- 太もも (upperLeg): X 負で前に上げる、X 正で後ろ。横方向は Z。
- 足踏み・歩く: 左右 upperLeg X を交互 (-30⇔+10)、膝連動、腕は逆位相、hips を小さく上下。
- 足首 (foot): つま先伸ばし = X 正 (+20〜40) / 反らす = X 負 (-20)。
- 膝を曲げる姿勢では必ず hips の p.y を下げる (浅く -0.05〜-0.1 / 深く -0.2〜-0.35)。
  下げないと足が浮いて見える。
